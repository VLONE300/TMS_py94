Мое приложение может:
    1. Получить список всех продуктов.
Для этого будет определена функция которая ничего не принимает на вход.
Возвращает ответ JSON, содержащий информацию о каждом продукте, включая идентификатор, название, белок, жиры и углеводы.
Пример[{"id": 1, "name": "Hamburger", "protein": 13, "fats": 9, "carbs": 30}]
	2. Создает новый продукт.
Ожидается тело запроса JSON, содержащее информацию о новом продукте, включая название, белки, жиры и углеводы.
Добавляет новый продукт в базу данных и возвращает ответ JSON с идентификатором, названием, белками, жирами и углеводами вновь созданного продукта.
	3. Обновляет существующий продукт.
Ожидается тело запроса JSON, содержащее обновленную информацию о продукте (название, белок, жиры, углеводы).
Находит продукт с указанным идентификатором, обновляет его атрибуты и возвращает ответ JSON с идентификатором, названием, белком, жирами и углеводами обновленного продукта.
Если продукт с указанным идентификатором не найден, возвращается ответ JSON, указывающий, что продукт не найден.
	3. Удалить Продукт.
Удаляет существующий продукт.
Находит продукт с указанным идентификатором, удаляет его из базы данных и возвращает ответ 204.
Если продукт с указанным идентификатором не найден, возвращается ответ JSON, указывающий, что продукт не найден.
